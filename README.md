# Zundamon's Weekend (Zundamon no Kyujitsu)

## Overview
"Zundamon's Weekend" (hereafter referred to as "this game") is a fan-made derivative work created by **Harunouta** (X/Twitter: [@shounokoto](https://x.com/shounokoto) / [@harunouta](https://x.com/Harunouta_sh)) featuring characters from the "[ZunZun Project](https://zunko.jp/)".

## Terms of Use （ず・ω・きょ）
This game and the contents of this repository may be used for both commercial and non-commercial purposes, provided that the following conditions are met:

**[Conditions]**

* Porting to other environments and printing are permitted, but **redistribution to third parties and resale are strictly prohibited.**
